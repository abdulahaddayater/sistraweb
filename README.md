# Sistra Diamonds

Luxury fine jewellery website for **Sistra Diamonds** — a Middle Eastern maison aesthetic inspired by houses such as Cartier, Tiffany & Co., Graff, Van Cleef & Arpels, and Bvlgari.

## Stack

- React 19 + TypeScript
- Vite
- React Router
- CSS Modules

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Notes

Product imagery uses curated Unsplash photography for demonstration. Replace with owned atelier assets for production.
